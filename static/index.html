Расширение для Chrome

Опционально: расширение скачивает YouTube прямо с вашего ПК. На сайте YouTube тоже работает без расширения.

Скачать расширение

Распакуй ZIP → chrome://extensions → Режим разработчика → Загрузить распакованное → папка extension

Зайдите на youtube.com и войдите в аккаунт. Расширение само возьмёт cookies и скачает видео с вашего компьютера — локальный сервер не нужен.
